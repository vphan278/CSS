# CSS Practice

This repository contains CSS practice exercises created while learning how to style web pages.

The examples demonstrate how CSS can control layout, colors, spacing, and visual appearance of HTML elements.

--------------------------------------------------

TOPICS COVERED

- CSS selectors
- Colors and fonts
- Box model
- Layout and spacing
- Styling HTML elements

--------------------------------------------------

GOAL

The goal of this repository is to practice styling web pages using CSS and understand how CSS works with HTML.
